📓 Learn ₿TC

🔐 Explore SHA256

Open Terminal and copy/paste:
perl -e "print qq(your text to hash here, case matters)" | shasum -a256
Examples:

  • The hash of 21 is 6f4b6612125fb3a0da0b33f3c7c8e29a3f4439cf567d51b6ec2d1e7e0b37a9e6
  • The hash of Bitcoin is b4056df6691f8dc72e56302ddad345d65fead3ead9299609a826e2344eb63aa4, while bitcoin is 6b88c087247aa2f07ee1c5956b8e1a9f4b1e1e2e6e2f711c7d6e6b7965a7ab6a

SHA256algorithm.com

≐ 2099999997690000

There will only ever be 2,099,999,997,690,000 Satoshis, written as 20,999,999.9769 ₿TC. The true number is actually not quite 21 million due to fixed-point arithmetic for binary division.

Why? Because the code uses fixed-point arithmetic for binary division.
Just a fun little fact for ya.

⚙️ Cybersecurity Recommendations

💻 Apps and Services
  • Password Managers: Bitwarden, Proton Pass, KeePassXC
  • 2FA: Ente Auth, Aegis
  • Email: ProtonMail
  • Aliases: SimpleLogin, Duck Duck Go
  • Maps: Organic Maps
  • VPN: Mullvad, Proton VPN
  • Browsers: Brave, Mullvad, Tor
  • Search: Duck Duck Go
  • Encryption: VeraCrypt, Cryptomator
  • File Sharing: Wormhole, Onion Share, Proton
  • Cloud Storage: Proton Drive, crypt.ee
  • Video Chat: Brave Talk, Jitsi Meet, Keet
  • PGP: GPG Keychain
  • Notes: Standard Notes, Notesnook
  • Chat: Matrix, Element, Session
  • Computers: System76, Purism
  • Desktop OS: Linux, Pop!_OS, Tails
  • Hardware Wallets: Coinkite Coldcard, Foundation Passport, Blockstream Jade
  • Phone: Graphene OS
🍎 Actions To Take Soon
  • Change all passwords to be unique and long
  • Cover your webcam
  • Always activate 2FA (not SMS)
  • Minimize apps installed, use web app shortcuts
  • Disable location and most app permissions
  • Disable voice-activated assistants
  • Use a VPN
  • Encrypt cloud backups with VeraCrypt or Cryptomator
  • Ditch Facebook & Google products
  • Try pseudonyms
  • Use email aliases (SimpleLogin, DuckDuckGo, Proton, Bitwarden)
